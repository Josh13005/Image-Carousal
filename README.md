# Image-Carousal
Simple Image Carousal using React, Node and Express.

## Installation

Use the package manager npm to install the dependencies on the:

Client
```bash
cd client
npm install
```

alternatively,
```bash
cd client
npm install react
npm install axios
```

server
```bash
cd server
npm install
```
alternatively,
```bash
cd server
npm install express
npm install nodemon
npm install cors
```
## Usage

1) Run the server
```bash
cd server
npm run dev
```
2) Run the client
```bash
cd client
npm start
```
